# Flow.Launcher plugins
I made som plugins for [Flow.Launcher](https://www.flowlauncher.com/).

## Plugins

### Private Internet Access
Enable/disable VPN and select region for [Private Internet Access (pia)](https://www.privateinternetaccess.com/) using [Flow.Launcher](https://www.flowlauncher.com/)

### Guid
Generate a random guid.
